Demo Music Box

This is a demo that shows you how to use the AudioPlayer class.
View the web console to see a console message on kick!

Enter a URL to an audio file:
Example:
https://yayinternet.github.io/hw4-music/songs/twice-knock-knock.mp3

See files:

  • demo.js: Demo class: Powers the demo (creates the AudioPlayer; handles the form)
  • script.js: Script that creates the Demo object.
